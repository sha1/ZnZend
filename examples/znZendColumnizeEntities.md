### Example for znZendColumnizeEntities

```php
<!-- In view script -->
<style>
  .border { border: 1px solid black; }
</style>

<?php
$entities = array();
for ($i = 0; $i < 8; $i++) {
    $obj = new stdClass();
    $obj->name = 'Object' . $i;
    $obj->number = mt_rand(1, 1000);
    $entities[] = $obj;
}

$params = array(
    'cols' => 3,
    'entities' => $entities,
    'entityCallback' => function ($entity) {
        $output = sprintf(
            '<div align="center">%s<br />Number: %d</div>',
            $entity->name,
            $entity->number
        );
        return $output;
    },
    'leftToRight' => true,
    'remainderAlign' => 'center',
    'tdClass' => 'border',
);
echo $this->znZendColumnizeEntities($params);
?>
```
